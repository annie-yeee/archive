# 명령 프롬프트

- 파워쉘 : 기능 업그레이드한 버전
- 명령어
    - dir : 목록 자세히 보기 ( 파일 존재 확인 )
    - dir /w : 목록 간략하게 보기
    - cd (change directory) 경로명 : 경로 옮기기
    - cls : 프롬프트 창 깨끗하게 clear
    - ⬆️ : history 전에 작성했던 것 불러오기
    - Tab : 자동완성
    - 마우스 오른쪽 : 복사 / 붙여넣기
    - Shift + Tab : 들여쓰기 했던 것 취소하고 한 단계 위로 올라가기

- 컴파일과 실행 방법
    - dir : 파일 존재 확인
    - javac -encoding utf-8 [파일명.java](http://파일명.java) : 컴파일
    - java 파일명 : 실행
